Instruções:

A ou D -> Walk
A+Shift ou D+Shift -> Run
Control (no solo) -> Roll
Control (no ar) -> Dash
(Segurar) W -> Attack D
(Segurar) S -> Attack C
J (no solo) -> Attack A
J (no ar) -> Attack E
H -> Kick
K -> Cast Fireball
L -> Shoot Arrow
B -> Dance
Space -> Jump A -> Jump B -> Jump C
Backspace -> Lay
